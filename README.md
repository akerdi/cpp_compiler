# CPP_Compiler

Implement `Input String` to `AStruct*`.

Usage:

```c++
string input;
input = "if (+ x y) { * 890909 9 } { / 9 3 }";
AStruct* program = compiler(input);
program->print();
program->deleteNode();
```
